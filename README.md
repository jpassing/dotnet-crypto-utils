# Crypto utility classes for .NET

`RSAExtensions` is a utility class that adds import and export functionality
for PEM-encoded RSA public keys to .NET Framework and older .NET Core versions.

See related blog posts:

* [What’s inside a PEM file](https://jpassing.com/2021/11/27/what-is-inside-a-pem-file/)
* [Encoding public keys in PEM format](https://jpassing.com/2021/11/30/using-pem-to-encode-public-keys/)
* [Importing RSA public keys in downlevel .NET and .NET Framework versions](https://jpassing.com/2021/12/05/importing-rsa-public-keys-in-downlevel-dotnet-and-dotnet-framework-versions/)
* [Exporting RSA public keys in .NET and .NET Framework](https://jpassing.com/2021/12/07/exporting-rsa-public-keys-in-dotnet-and-dotnet-framework/)
